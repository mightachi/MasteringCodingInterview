# ML Engineering Interview Preparation

This folder contains materials to prepare for an **ML Engineering** round focused on:

- **Inference platform expertise** – scalable, reliable, low-latency, cost-efficient model serving
- **ML platform expertise** – reproducible training, experimentation, lifecycle management
- **End-to-end MLOps** – from training to deployment and observability
- **Production Python** – clean code, library depth, technical communication
